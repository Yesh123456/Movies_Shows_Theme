
# Movies and Shows

This project is about creating a Movies and Shows theme.
## Installation

Install my-project as zip or clone.

```bash
    git clone https://github.com/Yesh123456/movies-shows.git
```

Install Sass.(For Ubuntu)

```bash
     sudo apt-get install -y sass 
```

## Features

- Colorfull Scheme
- Live previews
- Responsive app
- Web Application
